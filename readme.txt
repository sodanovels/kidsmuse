AGO KidsMuse Demonstration Project
Media Library Installation Readme
July 1998

The Kidsmuse Demonstration Project was designed to run on monitors capable of displaying thousands of 
colors.

You will need Adobe Acrobat Reader to view the Logbook that is designed to accompany the Kidsmuse 
demo.  You can download the reader from Adobe's website at http://www.adobe.com.

You can also experience the Kidsmuse Demonstration Project as an online project.  It is accessible on the 
AGO's website at http://www.ago.net/kidsmuse/

You will need Macromedia's Shockwave Plug-in and a current web browser.
